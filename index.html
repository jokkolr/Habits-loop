If you want to quit drugs, apply these reverse rules:

  • Make it Invisible: Avoid triggers and suppliers.
  • Make it Unattractive: List the harm done and read daily.
  • Make it Hard: Get accountability and remove access.
  • Make it Unsatisfying: Replace highs with healthy rewards.
